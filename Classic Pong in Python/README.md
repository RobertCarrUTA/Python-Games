# Pong in Python

## Compile Instructions
* Enter python pong.py into the terminal where the file is located

## Controls
*Controls can be changed, just CTRL + F (or something similar) **window.onkeypress(paddle_a_up, "w")***


Unedited controls are:
* Paddle A (left side)
  * Up: W
  * Down: S
* Paddle B (right side)
  * Up: Up Arrow
  * Down: Down Arrow
